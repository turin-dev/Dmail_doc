✈ 메일 보내기

Discord에서 바로 이메일을 작성하고 보내세요!

⚠
Gmail 전용 기능: 현재 메일 발송은 Gmail 계정에서만 지원됩니다. 네이버, 다음 등 IMAP 계정은 메일 수신 알림만 가능합니다.

📝 메일 보내기 튜토리얼

메일 작성 명령어 실행

Discord에서 다음 명령어를 입력하세요:

/메일 작성

또는 알림 메시지의 "답장하기" 버튼을 클릭해도 됩니다.

발신 계정 선택

여러 Gmail 계정이 연결되어 있다면, 보낼 계정을 선택하세요.

계정이 하나만 있으면 자동으로 선택됩니다.

메일 작성 패널 확인

봇이 DM으로 메일 작성 패널을 보냅니다.

패널에서 받는 사람, 제목, 본문을 확인하고 수정할 수 있어요.

내용 입력

✏ 버튼을 클릭해서 메일 내용을 입력하세요:

  • 받는 사람: 수신자 이메일 (쉼표로 여러 명 가능)
  • 제목: 메일 제목
  • 본문: 메일 내용 (마크다운 지원!)

첨부파일 추가 (선택)

첨부파일을 추가하려면 DM에 파일을 업로드하세요.

자동으로 작성 중인 메일에 첨부됩니다.

💡
첨부파일 제한: 최대 8개 / 총 20MB까지

보내기!

"보내기" 버튼을 클릭하면 메일이 발송됩니다.

✅
발송이 완료되면 "전송 완료" 메시지가 표시됩니다.

💬 작성 패널 예시

✉ 메일 작성 패널

보내기
  • 받는 사람: recipient@example.com
  • 보내는 사람: user@gmail.com

안녕하세요!

메일 본문 내용이 여기에 표시됩니다.

👥 여러 명에게 보내기

받는 사람 칸에 쉼표(,)로 구분해서 여러 주소를 입력하세요:

user1@example.com, user2@example.com, user3@example.com

각 수신자에게 동일한 메일이 발송됩니다.

💡 팁

  • 최신 패널 사용: 여러 개의 작성 패널이 있다면 가장 최근 패널만 사용하세요. 이전 패널은 작동하지 않습니다.
  • 첨부파일: 이미지, 문서, 압축파일 등 다양한 파일을 첨부할 수 있어요.
  • 발송 취소: 한 번 보낸 메일은 취소할 수 없으니 신중하게 확인 후 보내세요!

🔧 문제 해결

❓ "연결된 메일 계정이 없어요" 메시지가 나와요

Gmail 계정이 연결되어 있어야 합니다. /메일 추가로 Gmail을 연결해주세요.

❓ 메일 발송에 실패했어요

OAuth 인증이 만료되었을 수 있어요. /메일 관리에서 계정을 확인하고, 필요하면 다시 연결해주세요.

❓ 첨부파일이 추가되지 않아요

파일 크기가 20MB를 초과하거나, 이미 8개의 파일이 첨부된 경우 추가할 수 없어요.
