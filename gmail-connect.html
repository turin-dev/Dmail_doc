📧 Gmail 연결하기

Google 계정을 OAuth로 안전하게 연결하세요. 비밀번호 입력 없이 간편하게!

☀️ 🌙

🔒 OAuth란?

OAuth는 비밀번호를 공유하지 않고 안전하게 앱에 권한을 부여하는 방식입니다.

✅
Gmail OAuth의 장점:
  • 비밀번호를 D-Mail에 알려줄 필요 없음
  • 언제든지 Google 계정에서 권한 해제 가능
  • 메일 읽기/보내기 권한만 요청

📝 Gmail 연결 단계

메일 추가 명령어 실행

Discord에서 D-Mail 봇에게 다음 명령어를 입력하세요:

/메일 추가

또는 "메일 연결하기" 버튼을 클릭해도 됩니다.

Gmail 주소 입력

팝업 창에서 연결할 Gmail 주소를 입력하세요.

example@gmail.com

@gmail.com 또는 @googlemail.com 주소를 입력하면 자동으로 OAuth 방식으로 진행됩니다.

"메일 소유 인증하기" 버튼 클릭

봇이 보내는 메시지에서 "메일 소유 인증하기" 버튼을 클릭하세요.

버튼을 클릭하면 Google 로그인 페이지로 이동합니다.

Google 계정 로그인

Google 로그인 페이지에서:

  1. 연결할 Gmail 계정을 선택합니다
  2. D-Mail의 권한 요청을 확인합니다
  3. "허용" 버튼을 클릭합니다
⚠
"이 앱은 확인되지 않았습니다" 경고가 나타날 수 있어요.
이 경우 "고급" → "안전하지 않은 페이지로 이동" 을 클릭하세요.

연결 완료!

권한을 허용하면 자동으로 Discord로 돌아오고, 연결이 완료됩니다.

✅
연결 완료! 이제 새 Gmail이 도착하면 Discord DM으로 알림을 받게 됩니다.

🔧 문제 해결

❓ "앱이 확인되지 않았습니다" 경고가 나타나요

Google의 앱 검증 과정에 있어서 나타나는 정상적인 경고입니다. "고급" 클릭 후 "안전하지 않은 페이지로 이동"을 선택하세요.

❓ 인증 버튼이 작동하지 않아요

브라우저에서 팝업이 차단되었을 수 있어요. 팝업 차단을 해제하거나, 버튼 URL을 복사해서 직접 브라우저에 붙여넣기 해보세요.

❓ 연결 후 알림이 안 와요

연결 직후에는 기존 메일에 대한 알림이 오지 않습니다. 연결 이후에 도착하는 새 메일부터 알림이 발송됩니다.
