🔔 Email Notifications

Get real-time notifications via Discord DM when new emails arrive!

☀️ 🌙

⚡ How Does It Work?

Check for New Emails

D-Mail checks your connected mailbox for new emails every 30 seconds.

Send Notification

When a new email is found, a notification is sent to your Discord DM.

The notification includes the sender, subject, and a preview.

Auto-Extract Verification Codes

For emails containing verification codes, AI automatically extracts and displays the code!

🔑 Automatic Verification Code Extraction

D-Mail uses Google Gemini AI to automatically extract verification codes from emails.

Supported code types:

  • ✓ Sign-up / login verification codes
  • ✓ OTP (One-Time Passwords)
  • ✓ Password reset codes
  • ✓ Two-factor authentication (2FA) codes
💡
When a verification code is detected, it's displayed prominently as Code: XXXXXX. Just copy and use it right away!

Item Details

⚠
Note: Notifications may be slightly delayed depending on the email server status.

💡 Good to Know

  • No alerts for existing mail: You won't receive notifications for emails that already existed when you connected your account. Notifications are only sent for new emails that arrive after the connection.
  • DM must be enabled: If the bot's DM is blocked, you won't receive notifications. Make sure to enable DMs from server members in your Discord settings.
  • Multiple accounts: Even if you connect multiple email accounts, all notifications go to a single DM. The notification will indicate which account the email belongs to.
