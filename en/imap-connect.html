📩 Connect Other Email

Connect Naver, Daum, and other IMAP-supported email services using app passwords.

☀️ 🌙

✓ Supported Email Services

Email Service Support Status Notes
Gmail ✓ OAuth Supported See Gmail Connection Guide
Naver Mail ✓ Supported App password required
Daum/Kakao Mail ✓ Supported App password required
iCloud Mail ✓ Supported App password required
Yahoo Mail ✓ Supported App password required
Zoho Mail ✓ Supported App password required
ProtonMail ✕ Not Supported Requires Bridge app (remote connection not possible)
Outlook / Hotmail ✕ Not Supported Currently unsupported due to API costs
⚠
Notice for Outlook users: Outlook/Hotmail is currently not supported due to Microsoft API cost limitations. We recommend using a Gmail account instead.
⚠
Notice for ProtonMail users: ProtonMail requires the ProtonMail Bridge desktop app for IMAP access, which cannot be used with remote services like D-Mail.

🔑 What is an App Password?

An app password is a separate password that can only be used by a specific app, instead of your regular password.

💡
Benefits of app passwords:
  • No need to share your actual password
  • You can delete the specific app password anytime if needed
  • Your email service's security features remain intact

📗 Naver Mail App Password Guide

Go to Naver Mail Settings

Log in to Naver Mail and navigate to Settings.

Or click the link below:

🔗 Naver Mail Settings

POP3/IMAP Settings

Click the POP3/IMAP Settings menu.

Change IMAP/SMTP usage to Enabled.

Generate App Password

On the same page, click the "Generate App Password" button.

Enter D-Mail as the app name and generate it.

⚠
The generated app password is only shown once! Copy and save it right away.

Enter in D-Mail

Go back to Discord and click the "Enter App Password" button.

Enter the copied app password to complete the connection!

💛 Daum/Kakao Mail App Password Guide

Go to Daum Mail Settings

Log in to Daum Mail and navigate to Settings → IMAP/POP3.

🔗 Daum Mail

Enable IMAP

Enable IMAP usage.

Generate App Password

Go to Kakao Account → Security → App Password to generate one.

🔗 Kakao Account Security

📸 Yahoo Mail App Password Guide

Go to Yahoo Account Security

Log in to Yahoo Mail and navigate to Account Security.

🔗 Yahoo Account Security

Enable Two-Step Verification

You must have Two-Step Verification enabled before you can generate an app password.

Skip this step if you already have it enabled.

Generate App Password

On the security page, click "Generate app password".

Select Other app, enter D-Mail, and click Generate.

⚠
Copy the generated 16-character app password immediately — it won't be shown again.

Enter in D-Mail

Go back to Discord and click the "Enter App Password" button.

Enter the copied app password to complete the connection!

💼 Zoho Mail App Password Guide

Go to Zoho Account Security

Log in to your Zoho account and navigate to Security settings.

🔗 Zoho Account Security

Enable Two-Factor Authentication

You must enable Two-Factor Authentication (MFA) before generating an app password.

Generate App Password

In security settings, find the "App Passwords" section and click "Generate New Password".

Enter D-Mail as the app name and generate it.

💡
Zoho Mail's IMAP server is imap.zoho.com (port 993). D-Mail will detect it automatically.

Enter in D-Mail

Go back to Discord and click the "Enter App Password" button.

Enter the copied app password to complete the connection!

💻 Connect in D-Mail

Run the Mail Add Command

/mail add

Enter Your Email Address

Enter the email address you want to connect.

example@naver.com

Enter the App Password

Click the "Enter App Password" button and enter the app password you generated.

💡
Your app password is encrypted and stored securely.

Connection Complete!

✅
Connection complete! You will now receive Discord DM notifications when new emails arrive.

⚠ Important Notes

⚠
Email sending limitation: The email sending feature (/mail compose) is currently only supported for Gmail accounts. Naver, Daum, and other IMAP accounts can only receive email notifications.
