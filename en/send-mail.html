✈ Send Email

Compose and send emails directly from Discord!

☀️ 🌙
⚠
Gmail Only Feature: Email sending is currently supported only with Gmail accounts. Naver, Daum, and other IMAP accounts only support receiving email notifications.

📝 Send Email Tutorial

Run the Compose Command

Type the following command in Discord:

/mail compose

You can also click the "Reply" button on a notification message.

Select Sender Account

If you have multiple Gmail accounts connected, select the account you want to send from.

If you only have one account, it will be selected automatically.

Review the Compose Panel

The bot will send a Compose Panel via DM.

You can review and edit the recipient, subject, and body from the panel.

Fill in the Content

Click the ✏ button to enter your email content:

  • To: Recipient email address (comma-separated for multiple)
  • Subject: Email subject line
  • Body: Email content (markdown supported!)

Add Attachments (Optional)

To add attachments, upload files in the DM chat.

They will be automatically attached to the email you are composing.

💡
Attachment Limits: Maximum 8 files / 20MB total

Send!

Click the "Send" button to deliver your email.

✅
Once the email is sent successfully, a "Sent Successfully" confirmation message will appear.

  • To: recipient@example.com

✏

📃 Markdown Support

You can use markdown syntax in the email body!

Input Result
**bold** bold
*italic* italic
# heading Large heading
[link](url) Clickable link
`code` code style

👥 Multiple Recipients

Separate multiple email addresses with commas in the "To" field:

user1@example.com, user2@example.com, user3@example.com

The same email will be sent to each recipient.

💡 Tips

  • Use the latest panel: If you have multiple compose panels open, only use the most recent one. Previous panels will no longer function.
  • Attachments: You can attach various file types including images, documents, and archives.
  • Cannot cancel: Once an email is sent, it cannot be cancelled or recalled. Please review carefully before sending!

🔧 Troubleshooting

❓ "No connected email account" error

You need a Gmail account connected. Use /mail add to connect your Gmail account.

❓ Email sending failed

Your OAuth authentication may have expired. Check your account in /mail manage and reconnect if necessary.

❓ Attachments are not being added

Files cannot be added if the total size exceeds 20MB or if 8 files are already attached.
