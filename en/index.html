⚡ Quick Start

Follow these 3 simple steps to start using D-Mail:

Start a DM with D-Mail Bot

Send a DM to the D-Mail bot on Discord to start the onboarding process automatically. Or use the /start command.

Connect Your Email

Use the /mail add command to connect Gmail or other email accounts. Gmail connects securely via OAuth, while other providers use app passwords.

Receive Notifications!

Now you'll receive real-time notifications via Discord DM when new emails arrive. Verification codes are automatically extracted for your convenience.
