🚀 Getting Started

New to D-Mail? Follow this guide to get started easily!

☀️ 🌙

✉ What is D-Mail?

D-Mail is a bot that lets you manage your email from Discord.

  • ✓ Real-time notifications via Discord DM when new emails arrive
  • ✓ Automatic verification code extraction (OTP, auth codes, etc.)
  • ✓ Read and reply to emails directly from Discord
  • ✓ Supports Gmail, Naver, Daum, and more

📝 Getting Started Tutorial

Send a DM to the D-Mail Bot

Find the D-Mail bot on Discord and send it a DM (Direct Message).

Sending a DM will automatically start the onboarding process.

💡
If onboarding doesn't start, try typing the /start command.

Complete the Onboarding

The onboarding consists of 4 stages:

  • Stage 1: D-Mail introduction
  • Stage 2: Support server & help information
  • Stage 3: Database initialization (automatic)
  • Stage 4: Connect your first email

Just click the "Next" button to proceed through each stage.

Connect Your Email Account

At the last stage of onboarding, click the "Connect Email" button.

Or you can connect an email anytime using the /mail add command.

📧 Gmail Connection Guide 📩 Other Email Guide

Done!

Once your email is connected, you're all set to use D-Mail!

You'll receive Discord DM notifications whenever new emails arrive.

✅
Congratulations! You can now use all of D-Mail's features.

💻 Key Commands

Command Description
/start Start the onboarding process
/mail add Connect a new email account
/mail manage Manage connected email accounts
/mail compose Compose a new email
!ping Check bot response speed
!info View bot information
