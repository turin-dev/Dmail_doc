🔧 문제 해결

D-Mail 사용 중 문제가 발생했나요? 여기서 해결 방법을 찾아보세요.

☀️ 🌙

🔗 연결 문제

❓ OAuth 인증 버튼이 작동하지 않아요

원인: 브라우저에서 팝업이 차단되었을 수 있습니다.

해결:

  • 브라우저의 팝업 차단을 해제하세요
  • 버튼을 우클릭해서 URL을 복사한 후 직접 브라우저에 붙여넣기
  • 다른 브라우저로 시도해보세요

❓ "이 앱은 확인되지 않았습니다" 경고

원인: Google의 앱 검증 과정에 있어서 나타나는 경고입니다.

해결:

  1. "고급" 또는 "Advanced" 클릭
  2. "안전하지 않은 페이지로 이동" 클릭
  3. 계속 진행

❓ 앱 비밀번호로 연결이 안 돼요

확인사항:

  • IMAP이 활성화되어 있는지 확인
  • 앱 비밀번호가 올바르게 입력되었는지 확인 (공백 주의)
  • 계정 비밀번호가 아닌 앱 비밀번호를 사용했는지 확인
  • 2단계 인증이 활성화되어 있어야 앱 비밀번호 발급 가능

🔔 알림 문제

❓ 알림이 전혀 안 와요

확인사항:

  • Discord 설정에서 봇의 DM이 차단되어 있지 않은지 확인
  • 메일 계정이 "연동 완료" 상태인지 /메일 관리에서 확인
  • 연결 직후에는 기존 메일에 대한 알림이 오지 않습니다 (새 메일부터)

❓ 알림이 늦게 와요

원인: D-Mail은 30초마다 메일을 확인합니다.

정상 범위:

  • 보통: 30초 ~ 1분 지연
  • 서버 상태에 따라 최대 2-3분 지연 가능

❓ 일부 메일만 알림이 와요

가능한 원인:

  • 메일이 스팸함으로 분류된 경우
  • 메일 서버의 동기화 지연
  • 동시에 많은 메일이 도착한 경우 일부 누락 가능

🔒 OAuth / 인증 문제

❓ "Gmail 인증 정보를 확인할 수 없어요" 오류

원인: OAuth 토큰이 만료되었거나 권한이 취소되었습니다.

해결:

  1. /메일 관리에서 해당 계정 삭제
  2. /메일 추가로 같은 계정 다시 연결
  3. OAuth 인증 다시 완료

❓ 봇이 재인증을 요청해요

원인:

  • Google 계정 비밀번호 변경
  • Google 계정에서 앱 권한 취소
  • 장기간 미사용으로 인한 토큰 만료

해결: 봇이 보내는 인증 버튼을 클릭해서 다시 인증하세요.

✈ 메일 발송 문제

❓ "연결된 메일 계정이 없어요" 오류

원인: Gmail 계정이 연결되어 있지 않습니다.

해결: /메일 추가로 Gmail 계정을 연결하세요.

⚠
메일 발송은 Gmail 계정에서만 지원됩니다.

❓ 메일 발송에 실패했어요

확인사항:

  • 받는 사람 이메일 주소가 올바른지 확인
  • 첨부파일이 20MB를 초과하지 않는지 확인
  • OAuth 인증이 유효한지 확인 (재연결 필요할 수 있음)

❓ 첨부파일이 추가되지 않아요

제한사항:

  • 최대 8개의 파일
  • 총 용량 20MB 이하
  • 작성 패널이 있는 DM 채널에서만 파일 추가 가능

🤖 봇 응답 문제

❓ 봇이 응답하지 않아요

확인사항:

  • 봇이 온라인 상태인지 확인
  • 명령어가 올바른지 확인 (슬래시 명령어: /메일)
  • !ping으로 봇 상태 확인

❓ 버튼을 눌러도 반응이 없어요

가능한 원인:

  • 이전 세션의 오래된 버튼은 작동하지 않을 수 있음
  • 네트워크 연결 문제
  • Discord 앱 재시작 시도
