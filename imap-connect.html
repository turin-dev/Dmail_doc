⚠
Outlook 사용자 안내: 현재 Outlook/Hotmail은 Microsoft API 비용 문제로 지원하지 않습니다. Gmail 계정 사용을 권장드립니다.

🔑 앱 비밀번호란?

앱 비밀번호는 일반 비밀번호 대신 특정 앱에서만 사용할 수 있는 별도의 비밀번호입니다.

💡
앱 비밀번호의 장점:
  • 실제 비밀번호를 공유하지 않아도 됨
  • 필요시 언제든 해당 앱 비밀번호만 삭제 가능
  • 메일 서비스의 보안 기능 유지

📗 네이버 메일 앱 비밀번호 발급

네이버 메일 설정 페이지 접속

네이버 메일에 로그인 후, 환경설정으로 이동하세요.

또는 아래 링크를 클릭하세요:

🔗 네이버 메일 설정

POP3/IMAP 설정

POP3/IMAP 설정 메뉴를 클릭하세요.

IMAP/SMTP 사용을 사용함으로 변경합니다.

앱 비밀번호 생성

같은 페이지에서 "앱 비밀번호 생성" 버튼을 클릭합니다.

앱 이름에 D-Mail을 입력하고 생성하세요.

⚠
생성된 앱 비밀번호는 한 번만 표시됩니다! 바로 복사해서 저장해두세요.

D-Mail에 입력

Discord로 돌아가서 "앱 비밀번호 입력하기" 버튼을 클릭합니다.

복사한 앱 비밀번호를 입력하면 연결 완료!

💛 다음/카카오 메일 앱 비밀번호 발급

다음 메일 설정 접속

다음 메일에 로그인 후, 환경설정 → IMAP/POP3로 이동합니다.

🔗 다음 메일

IMAP 사용 설정

IMAP 사용을 활성화합니다.

앱 비밀번호 발급

카카오 계정 → 보안 → 앱 비밀번호에서 발급받습니다.

🔗 카카오 계정 보안

/메일 추가

이메일 주소 입력

연결할 이메일 주소를 입력합니다.

example@naver.com

앱 비밀번호 입력

"앱 비밀번호 입력하기" 버튼을 클릭하고, 발급받은 앱 비밀번호를 입력합니다.

💡
앱 비밀번호는 암호화되어 안전하게 저장됩니다.
