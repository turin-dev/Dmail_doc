⚡ 어떻게 작동하나요?

새 메일 확인

D-Mail은 30초마다 연결된 메일함에 새 메일이 있는지 확인합니다.

알림 전송

새 메일이 발견되면 Discord DM으로 알림을 보냅니다.

알림에는 발신자, 제목, 미리보기가 포함됩니다.

인증코드 자동 추출

인증 코드가 포함된 메일은 AI가 자동으로 코드를 추출해서 보여줍니다!

💬 알림 예시

noreply@example.com

user@gmail.com

✉ 회원가입 인증코드

Code: 123456

메일 보기 답장하기 웹에서 보기

위와 같은 형태로 알림이 옵니다. 버튼을 눌러 메일을 확인하거나 답장할 수 있어요.

🔑 인증코드 자동 추출

D-Mail은 Google Gemini AI를 사용해서 메일에서 인증코드를 자동으로 추출합니다.

지원하는 코드 종류:

  • ✓ 회원가입/로그인 인증코드
  • ✓ OTP (일회용 비밀번호)
  • ✓ 비밀번호 재설정 코드
  • ✓ 2단계 인증 코드
💡
인증코드가 감지되면 Code: XXXXXX 형태로 눈에 띄게 표시됩니다. 복사해서 바로 사용하세요!

⏳ 알림 주기

항목 내용
확인 주기 30초마다
알림 지연 최대 30초 ~ 1분
동시 연결 여러 메일 계정 지원
⚠
참고: 메일 서버 상태에 따라 알림이 약간 지연될 수 있습니다.

💡 알아두면 좋은 점

  • 연결 직후 알림: 메일 연결 직후에는 기존 메일에 대한 알림이 오지 않습니다. 연결 이후에 도착하는 새 메일부터 알림이 발송됩니다.
  • DM 허용: 봇의 DM이 차단되어 있으면 알림을 받을 수 없어요. Discord 설정에서 서버 멤버의 DM을 허용해주세요.
  • 여러 계정: 여러 메일 계정을 연결해도 모든 알림이 하나의 DM으로 옵니다. 어떤 계정의 메일인지 알림에 표시됩니다.
